Introducción al uso del Jupyter Notebook¶

El Jupyter Notebook (Cuaderno Jupyter) es una herramienta que se puede utilizar para facilitar la resolución de (algunos o muchos) problemas de las guías de trabajos prácticos de la asignatura Radiodiagnóstico y Radioterapia (FI-UNER). Su fortaleza radica en la posibilidad de combinar texto enriquecido (títulos, negrita, ecuaciones, etc) con código ejecutable, intercalar imágenes y videos, etc. Desde el mismo notebook se pueden realizar presentaciones animadas (con diapositivas), sin tener que duplicar el contenido en otro archivo con diferente formato. Además el notebook se puede exportar como PDF, entre otros formatos.

Lo vamos a utilizar con Python, que es un lenguaje muy popular en la comunidad científica, de muy facil sintaxis y legibilidad, y para el cual existe un vasto universo de librerías que facilitan la resolución de ecuaciones con unidades físicas y la graficación de resultados.

Hay varias formas distintas de utilizar el Jupyter Notebook. Hay servicios online como Google Colab (https://colab.research.google.com) que permiten su uso remoto, pero nosotros vamos a usarlo ejecutándolo localmente. En ambos casos se accede a través de un navegador. Para ello, tras ejecutar el programa (que puede instalarse en Linux, Windows o MacOS) se cargará un servidor con el cual nos podremos comunicar accediendo a la dirección http://localhost:8888

¿Qué es un archivo .ipynb?¶

  • La respuesta corta: cada archivo .ipynb es un cuaderno, por lo que cada vez que creas un nuevo cuaderno, se creará un nuevo archivo .ipynb.

  • La respuesta más detallada: Cada archivo .ipynb es un archivo de texto que describe el contenido de tu cuaderno en un formato llamado JSON. Cada celda y su contenido, incluidos los archivos adjuntos de imágenes que se han convertido en cadenas de texto, se enumeran junto con algunos metadatos.

Puedes editar esto tú mismo, ¡si sabes lo que estás haciendo!, seleccionando "Editar > Editar metadatos del cuaderno" en la barra de menú del cuaderno. También puedes ver el contenido de tus archivos de cuaderno seleccionando "Editar" en los controles del panel de control.

Sin embargo, la palabra clave aquí es "puedes". En la mayoría de los casos, no hay ninguna razón por la que debas necesitar editar manualmente los metadatos de tu cuaderno.

Celdas y Kernel¶

Hay dos términos bastante destacados que deberías notar y que probablemente sean nuevos para ti: las celdas y los 'kernel' son fundamentales tanto para comprender Jupyter como para entender por qué es algo más que un simple procesador de palabras.

  • Un Kernel es un "motor computacional" que ejecuta el código contenido en un documento de cuaderno.

  • Una celda es un contenedor para texto que se muestra en el cuaderno o código que es ejecutado por el kernel del cuaderno.

Modos de Edición y Comando¶

Al seleccionar una celda, ésta puede colocarse en modo de edición (el borde izquierdo que indica la celda activa será de color verde, y el mismo color para el resto del recuadro, tal como se ve la única celda de la imagen anterior) o en modo de comando (el color del borde izquierdo será celeste y el resto del recuadro gris). Para pasar entre un modo u otro se puede usar el teclado (Esc y Return) o el mouse (pulsando el puntero dentro de la celda o sobre el espacio vacío de la izquierda).

  • Modo comando: Pulsar tecla Esc o el espacio vacío de la celda
  • Modo edición: Pulsar Enter o el contenido de la celda

Atajos de teclado¶

Hay un enorme conjunto de atajos de teclado que facilitan el uso del entorno, pueden ver la lista a través del menú Help/Keyboard Shortcuts. También se pueden editar y agregar combinaciones de teclas mediante el menú Help/Edit Keyboard Shortcuts, por ejemplo es muy util definir un keybinding para mover las celdas hacia arriba y otro para moverlas hacia abajo. A través del menú Help también se accede a un tour educativo por las diferentes partes del entorno, ¡aprovéchenlo!

Ejecución de celdas¶

Cada celda se puede ejecutar con Shift+Enter (o con el botón Run). El navegador envía el contenido de la celda al servidor, quien procesa el código Python o Markdown y devuelve el resultado, que se imprime a continuación.

Al ejecutar una celda Markdown se procesan los códigos que identifican el texto enriquecido (negrita, listas, tablas, etc) y se muestra el texto con el formato adecuado. Si se ejecuta una celda Code se procesa el código Python y, de no mediar errores, se muestra el resultado.

El error más común es ejecutar texto markdown como si fuese código Python, lo que arrojará un SyntaxError: invalid syntax.

Trust notebook¶

https://jupyter-server.readthedocs.io/en/stable/operators/security.html#security-in-notebook-documents

La opción "Trust Notebook" en Jupyter Notebook permite marcar un cuaderno (notebook) como confiable después de ejecutar todo el código en él. Esto es útil en situaciones en las que ha descargado un cuaderno de una fuente externa y desea asegurarse de que no haya código malicioso antes de ejecutarlo. Cuando confía en un cuaderno, se supone que el código en él es seguro para ejecutar.

Sin embargo, el principal inconveniente al usar la opción "Trust Notebook" es:

  • Seguridad: Marcar un cuaderno como confiable significa que confía en todo el código que contiene. Si el cuaderno proviene de una fuente no confiable o contiene código que no comprende completamente, podría ejecutar código malicioso sin darse cuenta. Por lo tanto, debe ser cauteloso al confiar en cuadernos de fuentes no verificadas.

Por otra parte, las desventajas de los "untrusted notebooks" en Jupyter pueden ser:

  1. No se pueden ejecutar automáticamente: Cuando un cuaderno se marca como no confiable, Jupyter Notebook no permite ejecutar celdas de código automáticamente. Esto significa que, si tiene un cuaderno con muchas celdas de código, debe ejecutar cada celda manualmente, lo que puede ser tedioso y propenso a errores.

  2. No se pueden exportar a algunos formatos: Algunos formatos de exportación, como HTML, PDF o slides, pueden estar restringidos para cuadernos no confiables. Esto puede limitar su capacidad para compartir o presentar el contenido del cuaderno de la manera deseada.

Por lo tanto, recuerda verificar el contenido manualmente. Antes de ejecutar un cuaderno no confiable, debes revisar cuidadosamente su contenido y el código que contiene.

Markdown¶

Markdown es un lenguaje de marcado (markup language) ligero y fácil de aprender para formatear texto plano. Su sintaxis tiene una correspondencia de uno a uno con las etiquetas HTML, por lo que algunos conocimientos previos de HTML serían útiles, pero definitivamente no es un requisito previo.
